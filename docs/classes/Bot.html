Readonly login

login: ((this: Bot, password?: Password) => Promise<void>)
  • Defined in src/bot.ts:177

Type declaration

    • (this: Bot, password?: Password): Promise<void>

    • 会优先尝试使用token登录 (token在上次登录成功后存放在this.CLIENT.dir下)

      无token或token失效时：

      • 传了password则尝试密码登录
      • 不传password则尝试扫码登录

      掉线重连时也是自动调用此函数，走相同逻辑 你也可以在配置中修改reconn_interval，关闭掉线重连并自行处理

      Parameters

      • this: Bot
      • Optional password: Password

        可以为密码原文，或密码的md5值

      Returns Promise<void>
